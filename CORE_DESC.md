# Архитектура Альмы

Атомами являются не буквы, а слова  
Словарь очень маленький, вручную отсортированный по смыслам (пара сотен слов)  
Таким образом, можно выделить довольно большой контекст на весь диалог, оставляя нейронку в пределах любого железа (+- 300 слов)

Для каждого слова есть параметры (прошлое настоящее будущее) (множественное число | единственное число {подумать, может нужно оставить выяснение числа на лингвиста})

Также в словаре находятся служебные слова типа {player_name} {place_id} {from_player_hit} {to_player_hit} {run_away} etc...

Нейронка не выключается при завершении диалога, служебные команды передаются как предложения.  
Служебные команды могут как отправляться нейронкой, так и приниматься : ({from_player_hit} -> зачем ты меня ударил? -> {from_player_hit} -> {to_player_hit})  
Контроллер npc детектит выходные служебные команды и принимает решение на их основе.  
Входные соответственно приходят через контроллер npc и передаются в контекст.

## Нейросеть состоит из 5 основных компонентов:

- Семантический анализатор - один слой, который принимает raw слово и на выходе дает 2д вектор (нужен для доп сортировки и уменьшения дискретизации).
- Поведенческое ядро - нейросеть, которая содержит основной контекст и генерирует raw слова {предложениями либо отдельными словами, пока не знаю, скорее отдельными словами} из словаря (с временными указателями и числом) \[для улучшения мышления, можно добавить второй контекст размышления, датасет составить так, чтобы ядро само принимало решение об ответе в основной контекст\]
- Пирамида памяти:
    - Долговременный контекст - содержит примерное описание событий (просто словами из словаря), гораздо короче основного контекста.
    - Супер долговременный контекст - содержит очень примерное описание событий (особый словарь, который содержит не отдельные слова, а целые цепочки событий одним атомом), гораздо меньше долговременного контекста.
    - Маленькая нейросеть для генерации долговременного контекста (принимает на вход долговременный контекст и последние вытесненные данные из основного контекста (например каждые 50 слов), полностью заменяет долговременный контекст своим выходом).
    - Маленькая нейросеть для генерации супер долговременного контекста (принимает на вход долговременный контекст)
        - Каждое событие имеет коэффициент важности, его выдает нейросеть для генерации. Также она выдает коэффициент забывания события. Коэффициент забывания уменьшает коэффициент важности каждый тик. Когда приходит новое событие в контекст, ищется наименее важное событие и если событие менее важное, чем новое - новое перезаписывает его. Некоторые события из-за коэффициента не могут быть забыты никогда. Дубликаты событий не допускаются. (подумать состоит ли событие только из ключа или из ключ+переменная {убил : имя})
- Лингвист - маленькая нейросеть, которая получает на вход предложение и делает его человекочитаемым (добавляет склонения и т.д).
- Ведущий - маленькая нейросеть, которая придумывает варианты ответов игрока на основе выходного предложения.
- (Опционально) - модуль сна, для него будет собираться лог основного контекста (и контекста размышлений если он включен) и во время сна, поведенческое ядро будет обучаться на этих данных. Потенциально опасно для целостности системы и может привести к произвольным реакциям.
- (Опционально) - \[дельта-ритм\] контекст кольцевого буфера прогрессивного шума (перлин как пример) : буфер смещается каждый тик, дает ощущение времени и произвольные реакции. Основная сложность - составление данных для первичного обучения. После первичного обучения, модуль сна должен записывать этот контекст как входные данные.
- (Опционально, может не сработать) - модуль двух полушарий : создает дополнительное полушарие с общим контекстом и дельта-ритмом для первого. Полушария переключаются в зависимости от фазы дельта-ритма. Пока одно полушарие работает, другое находится в режиме сна и обучается. Открывает доступ к слабому обучению во время бодрствования. В теории должно создавать не осознанное внутреннее противоречие (критическое мышление на уровне структуры).
- (Опционально) - модуль случайных воспоминаний : во время спокойствия полушария, подмешивает случайное относительно важное воспоминание в контекст размышлений если полушарие ни о чем не думает. В зависимости от важности увеличивается шанс. Шанс уменьшается в зависимости от времени, последние по времени воспоминания с большей вероятностью будут подмешаны в контекст размышлений, чем даже травматичные давние. \[Опционально, может не сработать, возможны циклические всплески\] Содержит маленькую нейросеть, которая принимает на вход первые несколько слов контекста и генерирует на основе него токен события особого словаря супер долговременного контекста -> если событие есть в последнем слое долговременной памяти и полушарие ни о чем не думает - воспоминание подмешивается в контекст размышлений с некоторым шансом, который зависит от времени и важности воспоминания.
- (Опционально) - модуль оценки принятых решений : маленькая нейросеть, которая принимает на вход основной контекст и первый слой пирамиды памяти. Выдает оценку принятым решениям, оценка используется для обучения с подкреплением.
